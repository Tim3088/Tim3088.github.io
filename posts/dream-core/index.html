仓库地址: https://github.com/Tim3088/dream-core

概述#

梦核是一个基于 Java 的项目，旨在提供聊天以及游戏相关信息存档服务，项目结构设计为模块化，包含多个实体类、服务接口和实现类。

技术栈#

SpringBoot + MySQL + MyBatis

技术架构设计#

  • 分层架构：Controller → Service → DAO
  • 关键技术点：
    • 本地缓存数据，降低数据库压力
    • JWT 实现无状态认证
    • 接入AI实现对话功能

问题与解决方案#

  • 频繁查询数据库获取游戏物品属性
    • 解决方案：将物品表中的数据在服务启动时统一缓存到本地，直接从本地查询

收获与不足#

  • 收获：
    • 熟悉了SpringBoot项目的主要开发流程和相关代码规范
    • 排查问题的能力：提升了通过断点调试程序、发现问题的能力
    • 学会了如何在ubuntu上部署java项目（详见之前发布的 Ubuntu部署SpringBoot项目 博客）
  • 不足：
    • 测试覆盖不全
      • 未实现压力测试，性能瓶颈未提前暴露
    • 日志与监控不足
      • 关键链路日志缺失，排查线上问题耗时较长
      • 未对慢查询、异常请求设置实时告警
    • 容灾能力不足
      • 未实现多机房容灾，单点故障可能导致服务不可用
    • 资源浪费
      • 服务器资源配置过高，未根据实际负载动态缩扩容
