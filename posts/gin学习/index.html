query Link to heading

  1. Query 获取 key 对应的值，不存在返回空字符串 
    nameQuery := c.Query("name")
  2. DefaultQuery key 不存在时返回一个默认值 
    sexDefaultQuery := c.DefaultQuery("sex", "male")
  3. GetQuery 获取 key 对应的值，并且返回 bool 标识，标识成功或者失败 
     nameGetQuery, nameExist := c.GetQuery("name")
  4. QueryArray 获取 key 对应的值，值是一个字符串数组，不存在返回空字符串数组(null) 类似 Query 但是返回的是数组 
    nameQueryArray := c.QueryArray("name")
  5. GetQueryArray 获取 key 对应的值，并且返回 bool 标识，标识成功或者失败 
    hobbyGetQueryArray, hobbyExist := c.GetQueryArray("hobby")
  6. QueryMap 获取 key 对应的值，值是一个字符串 map[string]string，不存在返回空数组 
     userQueryMap := c.QueryMap("user")
